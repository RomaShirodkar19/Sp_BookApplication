# Spring Boot Book CRUD API

A simple REST API built using Spring Boot for performing CRUD operations on books.

## 🚀 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- H2
- Maven
- Lombok

## Features

- Add a Book
- Get Book by name
- Update Book
- Delete Book by Id

## Run Locally

```bash
git clone https://github.com/RomaShirodkar19/Sp_BookApplication.git
cd Sp_BookApplication
mvn spring-boot:run
